# birkelbach_OWASPJuiceShop

This project contains a simple OWASP Juice Shop inspired login page created for homework 3, part 2  for CSCE 477. The page is served by a small Node.js server and includes both client-side and server-side validation.

## Project Files

- `477homework3.js`: Main project file containing the Node.js server, HTML, CSS, JavaScript, and validation logic.

## Validation Rules

- Email cannot be empty
- Password cannot be empty
- Email must contain `@`
- Password must be at least 8 characters long

## How to Run

1. Run in your terminal:

```bash
node 477homework3.js
```

2. Open your browser and go to:

```text
http://localhost:3000
```

## Purpose

The purpose of this project is to demonstrate how a basic login form can be built with front-end checks and server-side validation. It is designed to support the Juice Shop security assignment by showing simple secure input handling practices.
